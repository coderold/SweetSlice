# SweetSlice 🍰

**SweetSlice** is a mini cake ordering app built with Flutter and Dart. Users can select cake flavors, sizes, add-ons, delivery options, and provide personalized messages. The app works on both mobile and web with a responsive, user-friendly interface.

## Features
- Text fields: Recipient name, Dedication message, Delivery address, Delivery instructions  
- Radio buttons: Cake flavor, Cake size, Payment method  
- Checkboxes: Add-ons  
- Dropdown: Delivery day selection  
- Buttons: Submit & Reset  
- Images: Cake flavor preview
